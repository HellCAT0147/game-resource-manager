# LootHelper — инвентарь S.T.A.L.K.E.R. 2 → «Выгодный хабар»

Связка из двух частей (игра v1.9, GOG build 89790, **не обновляется** → настроил один раз, работает всегда):

1. **Мод LootHelper** (UE4SS + Lua, папка `LootHelper/`) — читает предметы из
   тултипа инвентаря (hover-скан) и пишет `inventory_dump.json`.
2. **Панель LootPanel** (PowerShell WPF, папка `panel/`) — окно поверх игры:
   рейтинг «НА ВЫБРОС» (худшие ₽/кг — что выкинуть при перегрузе) и «ТОП»
   (лучшие ₽/кг), плюс автосвайп мышкой по сетке (F10).

Дамп также вливается в веб-приложение: я (Claude) вычитываю его в
`sync/inventory.json`, дальше в приложении «🎮 Синхронизировать из игры».

> 🗒️ **Просто запустить оверлей и не вспоминать клавиши → [ШПАРГАЛКА.md](ШПАРГАЛКА.md)**

## Установка (один раз)

1. **UE4SS:** [UE4SS updated (mods/1910)](https://www.nexusmods.com/stalker2heartofchornobyl/mods/1910)
   → содержимое в `d:\Games\STALKER2\Stalker2\Binaries\Win64\` (рядом с exe).
   Ключевое в пакете — `ue4ss\UE4SS_Signatures\GUObjectArray.lua`: без него
   PS-скан падает на 1.9. Проверка: в `ue4ss\UE4SS.log` строка `PS scan successful`.
2. **Мод:** папку `LootHelper` → в `...\Win64\ue4ss\Mods\`
   (`enabled.txt` внутри уже включает мод).
3. **Игра:** режим экрана — **«оконный без границ»** (иначе панель не видна поверх).
4. **Панель:** запуск `panel\start-panel.bat`. Калибровка (один раз): в игре
   открой инвентарь, наведи курсор на **центр левой-верхней ячейки** сетки → **Ctrl+F10**,
   на **центр правой-нижней** → **Ctrl+F11**. Размер сетки (столбцы×строки) и скорость —
   в `panel\panel-config.json` (по умолчанию 8×10, 120 мс/ячейка).
   REC-индикатор в шапке панели: ● REC мигает — запись включена, ○ rec — выключена.

## Ежедневный ритуал (≈15 секунд)

В игре: **I → F9 → F10 → F9**

| Клавиша | Кто ловит | Что делает |
|---------|-----------|------------|
| `F9` | мод | вкл/выкл запись (вкл = новый снимок с нуля; тултип ускоряется). ⚠ Перебинди быструю загрузку с F9 в настройках игры! |
| `F10` | панель | автосвайп: курсор сам пробегает по ячейкам **и сам листает колесом** весь инвентарь до конца |
| `F7` | мод | топ-10 ₽/кг в UE4SS.log (запасной просмотр) |
| `Ctrl+F10`/`Ctrl+F11` | панель | калибровка углов сетки |

Предметы на 2+ ячейки — не проблема: идём по ячейкам, повторы дедуплицируются.
Инвентарь не влезает в разворот (есть скролл)? Тоже не проблема: `F10` листает
колесом сам и останавливается, когда новых предметов не появляется. Перекрытие
рядов безвредно (дедуп), так что скорость листания (`scrollClicks` в
`panel-config.json`) можно крутить свободно — только не задирай так, чтобы
перескакивать целые ряды.

## Что в дампе

`inventory_dump.json`: имя (+`[износ%]` у снаряжения), цена за штуку, вес за
штуку, количество (вычисляется как вес-стака ÷ вес-штуки), флаг `gear`
(оружие/броня — панель по нему не предлагает их выкидывать).

## Технические заметки (кровью написано)

- Чтение текста игры: **только** `W_Text_C:GetText():ToString()`.
- **Писать текст в игру нельзя**: сигнатура `FText::FText(FString&&)` не найдена
  на 1.9 → любой путь создания FText из строки (конструктор, `Conv_StringToText`)
  роняет игру. Поэтому оверлей — внешнее окно, а не игровой виджет.
- Скрытые поля тултипа хранят **протухшие** значения → проверяй `IsVisible()`
  (баг «[93%] у батона»).
- Массовое чтение свойств пула виджетов → access violation, `pcall` не спасает.
  Читаем только объекты, которые игра сама держит живыми (`ItemTooltip`).
- `LoopAsync` переживает dofile-перезагрузку скрипта → цикл должен звать
  глобальный `_LOOT_TICK`, который обновляется каждым dofile (иначе зомби-цикл
  со старым кодом).
- Полностью автоматического чтения контейнера нет: у `ItemManager`/`GAManager`
  ноль отражённых свойств, у BP-виджетов ноль своих функций. Hover-скан — MVP.

## Файлы

| Файл | Назначение |
|------|------------|
| `LootHelper/enabled.txt` | флаг «мод включён» для UE4SS |
| `LootHelper/Scripts/main.lua` | загрузчик: хоткеи, dofile scan.lua (менять не нужно) |
| `LootHelper/Scripts/scan.lua` | вся логика; правится при работающей игре |
| `panel/LootPanel.ps1` | панель поверх игры + автосвайп (UTF-8 **с BOM**!) |
| `panel/start-panel.bat` | запуск панели |
| `panel/panel-config.json` | калибровка/настройки (создаётся сам) |
