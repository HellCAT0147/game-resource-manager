# 🗒️ Шпаргалка — запуск оверлея в игре

Быстрая памятка «что нажимать», чтобы не вспоминать каждый раз.
Полные детали и настройка с нуля — в [README.md](README.md).

---

## Каждый раз (когда сел играть)

1. **Запусти панель** — двойной клик по
   [`panel\start-panel.bat`](panel/start-panel.bat).
   Слева сверху появится тёмное окошко «☢ ВЫГОДНЫЙ ХАБАР».
2. **Игра — в режиме «оконный без границ»** (иначе панель не видна поверх).
3. Играй. Когда захотел снять инвентарь:

   ### В игре: **I → F9 → F10 → F9**

   | Шаг | Клавиша | Что происходит |
   |-----|---------|----------------|
   | 1 | `I` | открыл инвентарь |
   | 2 | `F9` | **старт записи** — в панели замигает **● REC** |
   | 3 | `F10` | курсор сам пробегает по сетке (~10 сек, не трогай мышь) |
   | 4 | `F9` | **стоп записи** — **● REC** гаснет |

   Панель наполняется прямо по ходу. Прокрутил инвентарь ниже — жми `F10` ещё
   раз (всё копится в один снимок, пока не выключишь запись вторым `F9`).

Готово — панель показывает **«НА ВЫБРОС»** (что бросить при перегрузе) и по
клику на `[выброс]/[топ]` — **«ТОП»** (что нести/продавать).

---

## Индикатор записи (в шапке панели)

- **● REC** мигает красным → запись включена
- **○ rec** тускло → выключена (можно спокойно играть)

---

## Один раз (уже сделано, если оверлей уже работал)

- Калибровка сетки: открой инвентарь, наведи на **центр левой-верхней** ячейки →
  `Ctrl+F10`, на **центр правой-нижней** → `Ctrl+F11`.
  Нужна только при смене разрешения/раскладки инвентаря.
- Быструю загрузку в игре стоит **перебиндить с F9** (Настройки → Управление),
  иначе старт записи будет пытаться грузить сейв.

---

## Если что-то не так

| Симптом | Что сделать |
|---------|-------------|
| Панель не видна поверх игры | режим экрана → «оконный без границ» |
| `F10` водит курсор мимо ячеек | подними `delayMs` в `panel\panel-config.json` (120 → 180) |
| Курсор бегает не по тем клеткам | перекалибруй: `Ctrl+F10` / `Ctrl+F11` |
| В панели «Дампа пока нет» | сделай хоть один цикл `F9 → F10 → F9` |
| `F9` грузит сейв | перебинди быструю загрузку в настройках игры |
| Панель зависла/криво | закрой (✕) и запусти `start-panel.bat` заново |
